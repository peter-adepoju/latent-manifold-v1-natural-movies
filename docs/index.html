The story

Use the chapters below to move through the project as a narrative, not just as a stack of results.

Why this project exists

Natural movies are messy in exactly the way real neural data are messy. The point of the project is to ask whether V1 population responses still reveal structure when the stimulus is natural, continuous, and temporally autocorrelated.

QuestionDoes mouse V1 organize natural-movie responses into a low-dimensional manifold?
MethodUse Allen Brain Observatory recordings, real frame features, and repeat-aware controls.
PrincipleKeep the evidence stronger than the storytelling.
Real stimulus features extracted from the Allen natural movie frames.

The evidence

The strongest result so far is not that the manifold is beautiful. It is that held-out-repeat V1 activity can predict real movie-derived features under careful evaluation.

Continuous feature decoding is the project’s clearest positive signal.

The anatomy of the cohort

The current public repository is intentionally honest about scope. It has the ingredients of a broader multi-session analysis, but only a narrow cohort is processed right now.

The project currently centers on one represented layer and one Cre line.

Where the evidence stops

This is where the project earns trust: it says what it does not yet know.

  • It does not yet support broad multi-session replication claims.
  • It does not yet justify strong layer-specific biological conclusions.
  • It does not yet establish a cross-session shared manifold.
  • It treats geometry as descriptive unless it survives nulls and generalization.
Geometry and stimulus alignment are interesting, but still modest and interpretive.

What comes next

The next chapter is scale: more sessions, more layers, more rigorous cross-session comparisons, and a deep-model alignment story that is strong enough to stand on its own.

  • Expand the cohort beyond the current three sessions.
  • Test whether the geometry survives cell-count matching and reliability correction.
  • Build the NeuroAI repo into a larger project hub over time.
  • Keep the website readable, scientific, and honest.
The project begins with the movie itself, not with a convenient abstraction.
